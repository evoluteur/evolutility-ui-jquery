Evolutility Documentation

Generic Web UI for CRUD and more.

Evolutility is a UI-models driven set of views which adapt to different data structures. For each entity, a single UI-model defines the UI for all views.

Views

A set of generic Backbone views to browse, edit, filter, export and chart Backbone models and collections of different data structures.
Views for one model:
Browse Edit Mini JSON
Views for a collection of many models:
List Cards Charts
Views for actions on a collection or a model:
Filter Export

UI-Models

A declarative language to configure the views and map fields on screen to attributes in your Backbone models.
object - field - panel - panel-list - tab

Controller

A controller and a toolbar to manage the views and make them act together as a Single Page App.

Code

Evolutility is using Javascript, HTML5, CSS3, with these open source libraries: Backbone, Underscore, jQuery, D3.js, Bootstrap, and some UI widgets: Select2, bootstrap-datepicker.

The code is available at GitHub under the AGPL V3 license.
